This command can be used to **mark** a message on a bulletin
[board](./board.md) that you wish to remember.

Usage: mark &lt;number&gt;

Messages marked like this are still displayed by
'[look](./look.md) board' even if they have been
[read](./read.md) (they are marked with a '\*' after the message
number). You can use 'unmark &lt;number&gt;' to remove the mark. 'mark last'
and 'unmark last' can also be used.

See also: [Reply](./Reply.md), [Tail](./Tail.md),
[Write](./Write.md)
