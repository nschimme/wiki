**General [skills](./Skill.md)** (also known as
**[ranger](./Ranger.md) skills**) refers to a set of skills that
players can learn at a ranger [guild](./Guild.md). Despite the
name, there is no ranger [class](./Class.md), and these skills can
be practiced by any class without penalty.
