Type change XML to get the following information between \&lt; \&gt;. This is
mostly used for those that use a [mapper](./mapper.md)

\&lt;prompt\&gt;...\&lt;/prompt\&gt; : MUME prompt \&lt;room\&gt;...\&lt;/room\&gt; : Room
region \&lt;name\&gt;...\&lt;/name\&gt; : Room name
\&lt;description\&gt;...\&lt;/description\&gt; : Room description
\&lt;terrain\&gt;...\&lt;/terrain\&gt; : Room terrain information
\&lt;magic\&gt;...\&lt;/magic\&gt; : Room Magic info (portal, detect magic, watched)
\&lt;exits\&gt;...\&lt;/exits\&gt; : Exits (autoexits or manual command) \&lt;movement
dir=direction/\&gt; : Movement in given dir \&lt;movement/\&gt; : Movement in an
unknown (or unapplicable) dir \&lt;tell\&gt;...\&lt;/tell\&gt; : tell
\&lt;say\&gt;...\&lt;/say\&gt; : say \&lt;narrate\&gt;...\&lt;/narrate\&gt; : narrate
\&lt;song\&gt;...\&lt;/song\&gt; : song \&lt;pray\&gt;...\&lt;/pray\&gt; : pray
\&lt;shout\&gt;...\&lt;/shout\&gt; : shout \&lt;yell\&gt;...\&lt;/yell\&gt; : yell
\&lt;emote\&gt;...\&lt;/emote\&gt; : emote \&lt;hit\&gt;...\&lt;/hit\&gt; : hit (given)
\&lt;damage\&gt;...\&lt;/damage\&gt; : damage (recieved) \&lt;weather\&gt;...\&lt;/weather\&gt;
: weather message

There will be more, and all of the above is subject to change.
