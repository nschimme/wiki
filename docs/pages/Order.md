To **order** a pet or a [charmed](./Charm.md) person to do
something. Example:

` > order cat say I speak animal!`
`The Cat says 'R fsiedaamimal!'`

` > order followers smile  [Order all charmed followers]`

Note that it is not possible to order [mobiles](./Mobile.md) to do
any of the following commands: [narrate](./Narrate.md),
[yell](./Yell.md), [shout](./Shout.md),
[tell](./Tell.md), [sing](./Sing.md),
[pray](./Pray.md) and [beep](./Beep.md). There are no
restrictions on what commands other players can be ordered to perform
though.

See Also: [Command](./Command.md), [Charm](./Charm.md)
