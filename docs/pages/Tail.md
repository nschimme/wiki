This allows you when reading a board to see only the last few messages,
rather than scrolling through the entire list. The syntax is:

Tail &lt;num&gt;

which displays the last &lt;num&gt; messages on the board. The default value
for &lt;num&gt; is one screenful - 21, for a standard terminal.

[Mail](./Mail.md)
