This is a general command to **change** one of your parameters.

Use 'change &lt;parameter&gt;', where &lt;parameter&gt; is one of the following:

`   ``ALERTNESS``      changes how much attention you pay to hidden people`
`   ANSIFILTER     turns removal of special characters in whoises on and off`
`   `[`CHARSET`](./Charset.md)`        selects which character set your terminal supports`
`   `[`CLASS`](./Change_Class.md)`          changes your class`
`   `[`COLOUR`](./Colour.md)`         highlights certain text in color to make it more noticeable`
`   `[`DESCRIPTION`](./Description.md)`    changes the description others see when they look at you`
`   `[`EDITOR`](./Change_Editor.md)`         selects your editor for writing text`
`   `[`EQORDER`](./Change_Eqorder.md)`        customise equipment display order`
`   HEIGHT         sets the number of lines before a page break for `[`PAGER`](./Change_Pager.md)
`   HTML           toggle to use the WWW-connection (disabled)`
`   `[`LANGUAGE`](./Language.md)`       selects another language to speak*`
`   `[`LINKDROP`](./Linkdrop.md)`       determines your connection's inactivity timeout`
`   `[`MOOD`](./Mood.md)`           changes how bravely you act when fighting`
`   PAGER          selects how you want long texts to be displayed`
`   `[`PASSWORD`](./Password.md)`       changes your password`
`   `[`SPELLCASTING`](./Change_Spellcasting.md)`   sets/shows default spellcasting effort`
`   `[`TITLE`](./Title.md)`          changes your title (only for heros and above); 'change`
`                  title none' removes the title**`
`   `[`WHOIS`](./Whois.md)`          changes the text people see when they use whois on you`
`   WIDTH          set screen width for output`
`   `[`WIMPY`](./Wimpy.md)`          lets you flee automatically when wounded`
`   WORDWRAP       toggles if you want long lines to word wrap`
`   `[`XML`](./XML.md)`            toggles XML-output on/off`

- Remember to use one of the "standard languages" (whiteside: Westron)
  if you want to be understood by all players when using the Forgotten
  Tales (also known as [narrating](./Narrate.md)).

<!-- -->

- Note that you can not change your title while suffering Mandos Sleep.

More parameters might be available, type 'change' to have a complete
list. Ainu see /help change.

Example: \> change colour fight \> change language westron
