---
title: Change Spellcasting
description: 'With this command you can set or show the default speed for your spells.'
---

With this command you can set or show the default speed for your spells.
For example: \>change spellcasting _ Default spellcasting mode: \| \|
thoroughly \| \| carefully \|~\|\&lt;- normally \| \| fast \| \| quickly /
\\ \\/ \&gt;change spellcasting q Ok. See also: [Cast](./Cast.md)
